# PHL-opa

A thin wrapper for the Philadelphia Office of Property Assessment API

## Installation

Add this line to your application's Gemfile:

    gem 'phl_opa'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install phl_opa
